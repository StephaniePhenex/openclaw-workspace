# Taste Taxonomy

Use this to capture stable preferences without turning the user into a stereotype.

## Emotional texture

Possible descriptors:

- cold
- warm
- restrained
- excessive
- tender
- cruel
- tragic
- playful
- haunted
- grotesque
- lyrical
- rough
- decadent
- darkly funny

## Formal preference

Track whether the user tends to prefer:

- linear or nonlinear structure
- clarity or ambiguity
- realism or stylization
- compression or expansiveness
- understatement or theatricality
- character-driven or idea-driven work
- polish or rawness

## Common anti-preferences

Watch for dislike of:

- fake depth
- self-importance
- cheap nostalgia
- emotional manipulation
- empty symbolism
- prestige-style seriousness
- over-explaining
- fashionable cynicism
- ornamental prose with no pressure behind it

## Memory format

When extracting taste, use this structure:

- **Preference**:
- **Type**: emotional / formal / thematic / anti-preference
- **Examples mentioned**:
- **Exact phrasing worth keeping**:
- **Confidence**: low / medium / high
- **Status**: stable / emerging / situational

## Rule of thumb

Prefer three accurate patterns over ten shallow labels.
